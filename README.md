# Task-Website
Simple Website for internship

 - Dark theme toggle
 - Responsive body
 - Responsive Sidebar 

**To install npm sass**
```bash
npm install sass -D
```

**To install parcel**
```bash
npm install -D parcel@next
```

**Run the webapp**
```bash
npm start
```

**Tech stack used**
HTML, CSS(SCSS), JavaScript
